Red de cuatro estaciones meteorológicas automáticas (EMA) distribuidas en Saladillo,
desarrollada para la EEST N°1 "Gral. Savio".

## Estaciones

| ID | Ubicación |
|---|---|
| EMA-EET | EEST N°1 "Gral. Savio" |
| EMA-CFR | Club de Field Racing |
| EMA-DC | Delegación Comunal |
| EMA-CS | Centro de Saladillo |

## Stack

- Scraping y limpieza de datos en Python
- Base de datos en Supabase
- Dashboard meteorológico v5.1
- CI/CD con GitHub Actions
- Análisis de microclima e isla de calor urbana
