# Coordenadas cartesianas

Sistema de coordenadas que utiliza ejes perpendiculares (x, y, z).

## En 2D

$$\vec{r} = x\hat{i} + y\hat{j}$$

## En 3D

$$\vec{r} = x\hat{i} + y\hat{j} + z\hat{k}$$
