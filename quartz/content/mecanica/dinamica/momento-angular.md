# Momento angular

Cantidad de movimiento rotacional. Análogo rotacional del [[momentum-lineal]].

## Fórmula

$$\vec{L} = \vec{r} \times \vec{p} = \vec{r} \times m\vec{v}$$

Para rotación alrededor de un eje fijo:

$$L = I\omega$$

Donde $I$ es el momento de inercia.

## Unidad

kg·m²/s

## Conservación

Si el torque neto externo es cero, el momento angular se conserva.

## Relacionado con

- [[momentum-lineal]] — Análogo lineal
- [[torque]] — Cambio de momento angular: $\vec{\tau} = \frac{d\vec{L}}{dt}$
- [[mcp]] — Movimiento circular
