# Componentes de un vector

Descomposición de un vector en sus partes a lo largo de los ejes coordenados.

## En 2D

$$A_x = A\cos\theta$$

$$A_y = A\sin\theta$$

## Módulo y dirección

$$A = \sqrt{A_x^2 + A_y^2}$$

$$\theta = \arctan\frac{A_y}{A_x}$$
