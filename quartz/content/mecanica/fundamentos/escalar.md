# Escalar

Una magnitud **escalar** se describe solo con un número y una unidad. No tiene dirección.

## Ejemplos

- [[masa]] — 70 kg
- [[temperatura]] — 25 °C
- [[rapidez]] — 100 km/h (el módulo de la velocidad)
- [[energia-cinetica]] — 500 J
- [[potencia]] — 60 W
- [[densidad]] — 1000 kg/m³

## Diferencia con [[vector]]

| Escalar | Vector |
|---------|--------|
| Solo módulo | Módulo + dirección + sentido |
| [[masa]] | [[velocidad]] |
| [[temperatura]] | [[fuerza]] |
| [[tiempo]] | [[aceleracion]] |
| [[energia]] | [[momentum-lineal]] |
