# Primera ley de la termodinámica

> La variación de [[energia-interna]] de un sistema es igual al [[calor]] recibido menos el [[trabajo]] realizado por el sistema.

$$\Delta U = Q - W$$

## Convención de signos

| Proceso | Signo |
|---------|-------|
| El sistema recibe calor | $Q > 0$ |
| El sistema cede calor | $Q < 0$ |
| El sistema realiza trabajo | $W > 0$ |
| Se realiza trabajo sobre el sistema | $W < 0$ |
