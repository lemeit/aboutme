# Vector

Un **vector** es una magnitud física que tiene módulo, dirección y sentido.

## Representación

$$\vec{A} = A_x \hat{i} + A_y \hat{j} + A_z \hat{k}$$

## Conceptos relacionados

- [[escalar]] — Magnitud sin dirección
- [[velocidad]] — Vector que describe el cambio de posición
- [[aceleracion]] — Vector que describe el cambio de velocidad
- [[fuerza]] — Vector que causa aceleración
- [[campo-electrico]] — Vector que describe la fuerza por unidad de carga
